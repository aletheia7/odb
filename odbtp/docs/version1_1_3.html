      ODBTP -- Version 1.1.3 Changes

Fixed Bugs
Fixed the problem in service that was causing a numveric overflow error when smalldatetime fields are converted to character format.

Fixed the PHP extension's mssql_bind() alias definition. The third argument is now forced to be passed by reference.

Fixed the C client library functions that would cause PHP client applications to crash when retrieving the name of a prepared parameter that was not bounded.
 

Support for building shared version of C client library
Support for building both shared and static versions of the C client library was added to the configure process. A Visual C++ project for building a Win32 DLL version of the C client library was also added to the package.
 
Native support for GUID fields and parameters
Support for reading and writing GUID (uniqueidentifier) data was added to the ODBTP server, client library and PHP extension. GUID field and parameter data can now be easily processed.
